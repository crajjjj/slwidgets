# Icon Gallery


The icons SL Widgets can display, grouped by source mod. **Stage icons** fill through 9 steps (0 = empty, 8 = full) as a value rises — only a few representative steps are shown here. **Toggle icons** simply appear when their condition is active and vanish when it clears.


!!! note
    Icons are shown on a checkerboard so transparency is visible. In-game they render     over the HUD at whatever size, position, and colour you set in the iWant Status Bars MCM.     Colours here are the defaults — see [Customization](customization.md) for colour presets and custom packs.
